# Auto366-mobile

初步完成Shell命令下使用Root权限提取答案
Todo：原生 Kotlin Android 应用，通过 Shizuku API 访问天学网应用数据，实现答案提取和缓存清理功能。

## 项目简介

Auto366-mobile 是一个辅助工具应用，主要功能包括：
- **答案提取**：通过 Shizuku 访问天学网应用数据，提取题目答案并显示在悬浮窗中
- **缓存清理**：清理天学网应用缓存和本应用缓存
